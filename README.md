# LoginApp

A simple Android application that displays a login screen with username and password fields.

## Tech Stack
- Kotlin
- XML
- Android Studio
- Gradle

## Project Structure
- `app/src/main/java/` – Contains Kotlin source files
- `app/src/main/res/` – Contains UI layouts and resources
- `app/src/main/AndroidManifest.xml` – App configuration

## Main Files
- `MainActivity.kt` – Entry point of the application
- `activity_main.xml` – Main UI layout file

## Features
- Login screen UI
- Clean and simple design

## How to Run
Open the project in Android Studio and run it on an emulator or physical device.
